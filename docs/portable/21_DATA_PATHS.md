# Data Paths

Project:
- `.env` — project env config
- `_auths/` — optional auth directory (default)

User home:
- `~/.kmi/state.json` — state (rotation index, counters)
- `~/.kmi/trace/trace.jsonl` — trace
- `~/.kmi/logs/kmi.log` — log
- `~/.kimi/config.toml` — Kimi CLI current account
- `~/.kimi/_auths` — optional auths dir
